PHP Function Parameters:
    A function in PHP may be defined to accept one or more parameters. 
    Function parameters are a comma-separated list of expressions inside 
    the parenthesis in front of the function name while defining a function. 
    A parameter may be of any scalar type (number, string or Boolean), an array, 
    an object, or even another function.

PHP Returning Values:
    A PHP function can have an optional return statement as the last statement 
    in its function body. Most of the built-in functions in PHP return a certain value. 
    For example the strlen() function returns the length of a string. Similarly, a user 
    defined function can also return a certain value.
